# 🎮 Simon Game (jQuery Project)

A browser-based implementation of the classic **Simon Game**, built to practice and understand core concepts of **jQuery**, DOM manipulation, and event handling.

This project was developed as part of *Angela Yu’s Full-Stack Web Development Course* on Udemy, with additional custom features implemented independently.

---

## 🚀 Features

* 🎯 Dynamic pattern generation (game sequence increases each level)
* 🎧 Sound and animation feedback for user interactions
* ❌ Game over detection with visual cues
* 🔄 **Restart button** to reset the game instantly
* 📜 **Instructions page** for better user guidance
* ⌨️ Keyboard-based game start and interaction
* 🧠 Memory-based gameplay logic

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* jQuery

---

## 📚 Learning Objectives

This project primarily focuses on:

* Understanding **jQuery selectors and methods**
* Handling **user events** (click, keypress)
* Managing **game state and logic**
* Working with **animations and sound effects**
* Debugging real-world issues (state bugs, event handling conflicts)

---

## ▶️ How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/simon-game.git
   ```

2. Navigate to the project folder:

   ```bash
   cd simon-game
   ```

3. Open `index.html` in your browser
